# Slay English Bot

Slay English Bot - это телеграм-бот для изучения английского языка с использованием тестов.

## Запуск бота

1. Установите зависимости:

```bash
pip install -r requirements.txt
Запустите бота:
bash
Copy code
python main.py
Команды бота
/start: Начать общение с ботом.
/start_test: Начать тест по английскому языку.
/stop: Остановить отправку новых слов.
Пример использования тестовых данных
Запустите бота с помощью команды /start.
Начните тест, введя команду /start_test.
Отвечайте на вопросы бота, выбирая варианты ответов.
Получайте уровень владения английским языком и новые слова каждые 10 секунд.
Тестовые данные
Вопросы
Where are you from?
How old are you?
Are you having a nice time?
Could you pass the salt please?
Yesterday I went __________ bus to the National Museum.
Варианты ответов
["I’m France.", "I’m from France.", "French.", "I’m French."]
["I have 16.", "I am 16.", "I have 16 years.", "I am 16 years."]
["Yes, I’m nice.", "Yes, I’m having it.", "Yes, I am.", "Yes, it is."]
["Over there.", "I don’t know.", "Help yourself.", "Here you are."]
["on", "in", "by", "with"]
Правильные ответы
[1, 2, 3, 2, 1]
